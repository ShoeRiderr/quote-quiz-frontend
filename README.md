# Quote quiz frontend

This app use backend logic from [This project](https://github.com/ShoeRiderr/quote_quiz_api)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
